Classification

Once the text inside each document is cleaned, we applied a topic model to obtain a categorization of documents based in the most frequent words present in them. The topic model used was LDA (Latent Dirichlet Allocation) for python.

LDA is a hierarchical Bayesian model that allows us to create topics from different texts based in the top words present within the texts. We supervised manually the topic model and we created 5 topics:

  1. Topic 0 - Environment and territory
  2. Topic 1 - Health
  3. Topic 2 - Employment and Wellbeing (Social Security)
  4. Topic 3 - Legal matters
  5. Topic 4 - Audiovisual

With the topic model done, we can apply a machine learning technique to classify new documents. We used both Support Vector Machine (SVM) and Logistic Regression (LR). SVM of Multiclass learning was the model that had better performance.
